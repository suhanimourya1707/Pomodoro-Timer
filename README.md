# Interactive Web Project

## Description

This is an interactive front-end web project developed using **HTML, CSS, and JavaScript**. The project focuses on creating a dynamic user interface with customizable background images, integrated sound effects, and a clean, non-scrollable full-screen layout. It is lightweight, responsive, and ideal for learning JavaScript-based interactivity.

## Features

*  Change background images based on user selection
*  Preloaded sound effects for smooth playback
*  Full-screen, non-scrollable layout
*  Single background image (no tiling)
*  Interactive and user-friendly interface
*  Responsive design

##  Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling, layout, and background control
* **JavaScript** – Logic, DOM manipulation, and interactivity

## Purpose of the Project

* Practice HTML, CSS, and JavaScript fundamentals
* Learn DOM manipulation and event handling
* Build interactive UI features
* Suitable for college mini-projects and portfolios
